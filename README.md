# Linear Equations Assignment

Static Algebra I assignment app for a class roster. The student page lets each student choose from multiple assignments, including 30 linear-equation problems and 15 systems-of-equations problems with instant feedback. The teacher dashboard is a separate page that filters browser-local submissions by assignment.

Note: the dashboard stores submissions in the browser running the site. It is useful for testing the workflow, but it is not a shared online grade database yet.
