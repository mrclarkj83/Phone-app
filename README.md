# Linear Equations Assignment

Static Algebra I assignment app for a class roster. The student page lets each student enter their 9-digit student ID to access multiple assignments, including 30 linear-equation problems and 15 systems-of-equations problems with instant feedback. The teacher dashboard is a separate page that filters browser-local submissions by assignment.

Note: the dashboard stores submissions in the browser running the site. It is useful for testing the workflow, but it is not a shared online grade database yet.

The student ID gate is intended as a lightweight access check for a static GitHub Pages site. It does not replace a server-side login system.
